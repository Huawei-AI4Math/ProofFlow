Informal theorem and proof
Theorem: Prove that for all positive real numbers $a, b, c$, the following inequality holds: $a^{4}+b^{4}+c^{4} \geq a b c (a+b+c)$. Proof: We want to prove that $a^{4}+b^{4}+c^{4} \geq a b c (a+b+c)$ for all $a, b, c \in \mathbb{R}^{+}$. We know that for any real numbers $x, y, z$: $(x-y)^2 + (y-z)^2 + (z-x)^2 \geq 0$ $2x^2 + 2y^2 + 2z^2 - 2xy - 2yz - 2zx \geq 0$ $x^2 + y^2 + z^2 \geq xy + yz + zx$ Let $x=a^2, y=b^2, z=c^2$. $a^4 + b^4 + c^4 \geq a^2b^2 + b^2c^2 + c^2a^2$ Let $x=ab, y=bc, z=ca$. $(ab)^2 + (bc)^2 + (ca)^2 \geq (ab)(bc) + (bc)(ca) + (ca)(ab)$ $a^2b^2 + b^2c^2 + c^2a^2 \geq ab^2c + bc^2a + ca^2b$ $a^2b^2 + b^2c^2 + c^2a^2 \geq abc(b+c+a)$ Combining the two inequalities: $a^4 + b^4 + c^4 \geq a^2b^2 + b^2c^2 + c^2a^2 \geq abc(a+b+c)$ Therefore, $a^{4}+b^{4}+c^{4} \geq a b c (a+b+c)$.
